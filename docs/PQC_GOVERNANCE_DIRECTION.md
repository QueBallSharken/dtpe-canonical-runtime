DTPE Post-Quantum Governance Direction

Purpose

This document defines the architectural requirement that DTPE remain
post-quantum-ready through policy-governed cryptographic agility.

Core principle

The governance model must remain independent from any single
cryptographic algorithm family.

Governance decisions are made at the deterministic execution boundary.

Cryptographic profiles bind and prove those decisions.

Requirements

• cryptographic profiles must be policy-governed
• receipts must carry explicit profile identity
• migrations between profiles must be governed and evidenced
• verification must remain offline and independent
• historical governance evidence must remain reconstructable across profile generations

Strategic direction

DTPE is intended to preserve machine-verifiable governance evidence
across changing cryptographic assumptions, including post-quantum transitions.
